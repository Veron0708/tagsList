Файлы с расширением .html открываются в браузере.

  • Назначение текста
  • Выделение текста
  • Графические элементы
  • Структура страницы

Справочник тегов.

1. Теги для текста

  • p - абзац от английского paragraph
  • h1..h6 - заголовки от английского heading
  • ul/ol - маркированный/нумерованный список от английского unordered/ordered list li - пункт списка от английского list item

2. Теги для выделения списка

  • b - жирный от английского слова bold
  • i- курсивный от английского слова italic
  • del - перечеркнутый от английского delete

3. Теги для графических элементов

  • hr - линия от слова английского horizontal rule
  • img - картинка от английского image

4. Теги задающие структуру страницы

  • !DOCTYPE html - не тег а инструкция для браузера
  • html - заключает в себе все содержимое веб страницы
  • head - для хранения служебной информации о страницы
  • body - содержимое документа html
  • header - раздел шапки веб-страницы
  • nav- тег для создания меню
  • main - основной контент веб-страницы
  • section - задает раздел веб страницы
  • footer - раздел подвала веб страницы
  • div - универсальный контейнер
